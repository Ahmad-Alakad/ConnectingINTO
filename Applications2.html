Connecting INTO has an open API which allows your clients and parteners enhance their cloud intranets solution through adjusting workflows and business logic inside their cloud-based portals. Connecying INTO also represents a great SaaS platform for creating custom businesses applications.

The Connecting INTO API is open for:

  • CRM
  • Socil network groups(workgroups, projects)
  • Data storage (information blocks)
  • Notifications and Activity Stream
  • Tasks
  • Users and Departments
  • Calenders
  • Telephony

How to create your app for Bitrix24?

If you are an IT-specialist and considering creating a web app for Connecting INTO, please follow these simple instructions:

  1. If you'd like your application to be avaliable to all Connecting INTO accounts, become our partner by filling out this form. if you want to create a private app for your account only, registration is not necessary.
  2. Learn all about Bitrix24 API and how you can create your web apps.
  3. Select 'ADD application' in the Applications menu inside your Connecting INTO account (on the left) to upload your private app or register your web app in your partener profile if you want your ap to be listed publicly!

Web app authorization and security

The process of adding a new app to your Connecting INTO CLoud accounts is easy , and we also tried to make it as transparent and secure as possible. Each web app provides you with a list of data it may use (and change) in your Connecting INTO account, and asks your permission to do so. The new app will be installed only if you give your permission.

The oAuth security mechanism makes sure that this web app gets access only to the data you have specified (e.g. a web app may ask your permission to access taskes, CRM leads, etc).
